Blog 3: CSS Animations

For my third blog post, I will be writing about CSS animations. CSS animations can be used to change the styling of an element without using Javascript. With CSS animations, you can change any CSS property you want. Animations are commonly used for changing colors or making something grow or shrink, but they can do more complex things like make elements move across a page. To use CSS animations, you have to specify keyframes that determine what styles an element will have at certain times. Here is an example of a keyframe that will changes the color of the h1 tag on this page: 

      
        @keyframes example {
          from { color: blue; }
          to { color: red; }
        }

        #h1-animation {
          color: red;
          animation-name: example;
          animation-duration: 4s;
        }

In this example, we set up a keyframe that will start the element's color at blue and end the element's color at red. Then in #h1-animation, we specify that we want the example keyframe to be applied by using the animation-name property. Finally, we say that the animation should take 4s to complete by using the animation-duration property. Here is a more complex animation:

The code for this animation is as follows:

      
        @keyframes example2 {
          0% { background-color: red; left: 0%; }
          25% { background-color: yellow; left: 25%; }
          50% { background-color: blue; left: 50%; }
          75% { background-color: green; left: 75%; }
          100% { background-color: red; left: 100%; }
        }

        #box {
          width: 100px;
          height: 100px;
          background-color: red;
          position: relative;
          animation-name: example2;
          animation-duration: 4s;
          animation-iteration-count: infinite;
        }

In this example, we are more specific with the stages of the animation. Properties for the animation are set when the animation is 0% finished, 25% finished, 50% finished, 75% finished, and 100% finished. This example uses background color as well as positioning to move the item. Finally, the animation does not stop because animation-iteration-count is set to infinite. If you only wanted the animation to run twice, you could set this property to 2.
